# Professional Event 2025

A simple event webpage built using HTML, CSS, and JavaScript. It displays event information and includes a sample registration button that shows a confirmation message when clicked.

## Features

- Displays event details
- Responsive user interface
- Registration button with a confirmation message
- Modern gradient background
- Clickable email and phone links

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How to Run

1. Clone the repository.

```bash
git clone https://github.com/NavinCyberAi/Event-Page.git
```

2. Navigate to the project folder.

```bash
cd Event-Page
```

3. Open `index.html` in your web browser.

## How to Use

1. Open the webpage.
2. View the event details.
3. Click the **Register Now** button.
4. A confirmation message will be displayed, and the button text will change to **Registration Completed**.
